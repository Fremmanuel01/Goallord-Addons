Premium, editorial-grade Elementor widgets. First widget in the suite: Announcement & News.

== Description ==

Goallord Addons is a curated set of premium Elementor widgets. Every widget is built for clean editorial design, performance, and deep customization — no bloat, no filler.

**Announcement & News widget:**

* Five polished layouts — Editorial Grid, Minimal List, Featured + Grid, Sidebar Bulletin, Timeline
* Fully dynamic repeater — image, category, date, author, title, description, badge, CTA, link
* Every element is styleable — typography, color, spacing, border, radius, shadow, hover states
* Responsive controls for columns, spacing, alignment
* Intentional entrance animations (fade, fade-up, fade-down, zoom, slide) with stagger, delay, duration, easing
* Hover micro-interactions — card lift, image zoom, CTA arrow shift
* Performance-first: IntersectionObserver, CSS variables, no heavy libraries
* Accessibility-ready: honors prefers-reduced-motion, ARIA roles, lazy-loaded images

== Installation ==

1. Upload the `goallord-addons` folder to `/wp-content/plugins/`.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Edit any page with Elementor. The new category **Goallord Addons** will appear in the widget panel.
4. Drag the **Announcement & News** widget onto your layout.
